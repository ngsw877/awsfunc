## awstk cleanup all

S3バケットとECRリポジトリを横断削除

### Synopsis

指定した文字列を含むS3バケットやECRリポジトリを一括削除するコマンドです。
CloudFormationスタック名を指定することで、スタック内のリソースを対象にすることもできます。

例:
  awstk cleanup all -k "test" -P my-profile
  awstk cleanup all -S my-stack -P my-profile
